# AzureLocalizandoServicos
Repositório para descrever o conteúdo aprendido no Lab da DIO referente à iniciação do ambiente Azure

Após criada a conta é possível já na home page verificar diversos serviços oferecidos pela Azure. A conta free ou trial pode ser limitada em determinados serviços ou regiões.
É possivel através dos setings trocar o idioma ou personalizar a aparência do seu Azure. É possivel personalizar os meuns, cores e outros.
Na opção "todos os serviços" há a apresnetação de tudo que a Azure oferece.

  Por exemplo conteudos da Computação, Redes, Armazenamento, Analises, IA + Machine Learning.

  Quando algum serviço está em versão previa, é necessário tomar cuidado, pois os serviço pode ser descontinuado.
